# AI-Assisted-Visual-Assessment-of-Physiological-Disorders-in-Sweet-Potato-Roots
Prompt Engineering and Command Chaining for AI-Assisted Visual Assessment of Physiological Disorders in Sweet Potato Roots in Climate Change Conditions

Overview

This repository provides a reproducible computational framework for the AI-assisted visual assessment of physiological disorders in sweet potato (Ipomoea batatas) roots, using prompt engineering and command chaining strategies.

The approach integrates:

* Structured prompt engineering
* Image-based evaluation workflows
* Automated analysis pipelines in Python

The framework is designed to support phenotyping, ecophysiological studies, and climate change impact assessments in sweet-potato crops.

---

Objectives

* Develop a structured workflow for AI-assisted visual diagnosis of root disorders
* Apply prompt engineering techniques to improve consistency and interpretability
* Support climate-resilient agricultural research through scalable analysis
* Provide a reproducible pipeline for scientific and technical applications

---

Conceptual Approach

The methodology is based on three main components:

1. Prompt Engineering

   * Structured prompts for standardized visual interpretation
   * Reduction of subjectivity in AI-assisted assessments

2. Command Chaining

   * Sequential execution of analytical steps
   * Modular and reproducible workflow

3. Python-Based Automation

   * Integration of scripts for data processing and analysis
   * Reproducibility and scalability

---

Description of files

* `main_analise.py` → Main analysis pipeline
* `prompt_engine_batata_doce_raizes.py` → Prompt engineering logic and definitions
* `gerar_hash.py` → Hash generation for reproducibility and data integrity
* `PDF` → Technical/scientific documentation of the methodology
